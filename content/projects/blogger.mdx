# Blogging Platform

This project is a full-fledged blogging platform where users can write, edit, and publish blog posts. It includes features for managing blog comments, likes, and user subscriptions.

## Core Features
- **Blog Creation**: Users can create and edit blog posts with rich text formatting.
- **Commenting & Liking**: Readers can leave comments and like posts.
- **User Subscriptions**: Users can subscribe to blog authors and receive email notifications.

## Technical Stack
- **Frontend**: Next.js for server-side rendering and SEO optimization.
- **Backend**: PostgreSQL with Prisma ORM for managing database records.
- **File Storage**: Cloudinary for handling image uploads and media.
- **Email Notifications**: SendGrid for email alerts to subscribers.
