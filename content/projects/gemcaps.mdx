# Gemcaps Healthcare RCM Platform

## Project Overview

Gemcaps is an enterprise-grade healthcare Revenue Cycle Management (RCM) platform that revolutionizes medical practice operations. The platform seamlessly integrates medical coding, billing, and patient management into a unified workflow powered by modern cloud technologies.

## Core Features

### Medical Coding Automation

The platform implements an advanced medical coding system with the following capabilities:

- Real-time ICD-10 and CPT code validation using a comprehensive rules engine
- Machine learning-powered code suggestions based on clinical documentation
- Integration with AHIMA certified coding guidelines
- Automated code review workflow with 98% accuracy validation
- Custom rules engine for specialty-specific coding requirements
- Historical coding pattern analysis for optimization

### Billing Management System

The billing module streamlines the entire revenue cycle with:

- Automated claim generation and submission to multiple clearinghouses
- Real-time payment processing integration with major payment gateways
- Automated ERA (Electronic Remittance Advice) processing
- Customizable billing rules and fee schedules
- Integrated payment plans and automated collections
- Comprehensive reporting and analytics dashboard

### Patient Services Portal

A feature-rich patient engagement system including:

- Automated insurance eligibility verification
- Prior authorization management with status tracking
- Patient self-service portal for appointments and payments
- Secure document exchange and digital forms
- Automated appointment reminders via SMS/email
- Real-time benefits explanation and cost estimates

### Database Schema

The PostgreSQL database schema is managed using Drizzle ORM with key tables:

```typescript
// Core schema definitions
const patients = pgTable('patients', {
  id: uuid('id').primaryKey().defaultRandom(),
  firstName: text('first_name').notNull(),
  lastName: text('last_name').notNull(),
  dateOfBirth: date('date_of_birth').notNull(),
  insuranceInfo: jsonb('insurance_info'),
  createdAt: timestamp('created_at').defaultNow(),
  updatedAt: timestamp('updated_at').defaultNow(),
});

const appointments = pgTable('appointments', {
  id: uuid('id').primaryKey().defaultRandom(),
  patientId: uuid('patient_id').references(() => patients.id),
  scheduledTime: timestamp('scheduled_time').notNull(),
  serviceType: text('service_type').notNull(),
  status: text('status').default('scheduled'),
  notes: text('notes'),
});
```

## Getting Started

### Prerequisites

```bash
node >= 18.0.0
npm >= 9.0.0
PostgreSQL >= 15.0
```

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/gemcaps.git

# Install dependencies
cd gemcaps
npm install

# Set up environment variables
cp .env.example .env.local

# Run database migrations
npm run db:migrate

# Start development server
npm run dev
```

### Environment Configuration

Required environment variables:

```
DATABASE_URL=postgresql://user:password@localhost:5432/gemcaps
AWS_ACCESS_KEY_ID=your_access_key
AWS_SECRET_ACCESS_KEY=your_secret_key
NEXT_PUBLIC_API_URL=http://localhost:3000/api
```

For additional setup instructions and configuration options, please refer to the project's GitHub repository.
