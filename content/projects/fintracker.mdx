# Personal Finance Tracker

This personal finance tool helps users manage their finances by tracking income, expenses, and budget. It provides data visualization to give a clear picture of spending habits and financial health.

## Core Features
- **Income & Expense Tracking**: Record and categorize income and expenses.
- **Budgeting**: Set up monthly budgets and track progress.
- **Financial Visualization**: View interactive charts of financial data.

## Technical Stack
- **Frontend**: Vue.js for building a reactive user interface.
- **Database**: Firebase for real-time data storage and user authentication.
- **Visualization**: Chart.js for rendering financial charts and graphs.
- **Styling**: Tailwind CSS for responsive design.
